# 职业卫生智能判定助手 / Occupational Hygiene AI Assistant

 让标准规范"开口说话" —— 用 AI 将复杂的职业卫生标准转化为人人可懂、秒级可判的智能工具。

## 痛点 (The Problem)

职业卫生检测与评价工作中，标准规范存在以下痛点：

| 痛点 | 场景 |
|------|------|
|  **标准繁杂** | GBZ 2.1、GBZ 2.2、GBZ 159、GBZ 331、GBZ/T 189、GBZ/T 192 等数百部标准交叉引用 |
|  **检索困难** | 一条判定需要翻阅多本标准，耗时耗力 |
|  **计算复杂** | 等效声级、不同工时的判定、工频电磁场、等计算容易出错 |
|  **表述晦涩** | 标准条文专业性强，一线人员理解门槛高 |
|  **判定主观** | 不同人员对同一结果可能得出不同结论 |

### 核心能力

- ** 标准智能解析**：自动提取标准中的限值、公式、判定条件
- ** 自动计算判定**：输入检测数据，自动计算等效声级、通风自动划分圆或者方块等
- ** 自然语言问答**：用大白话问标准，得到准确答案
- ** 报告辅助生成**：一键生成判定结论与依据说明
