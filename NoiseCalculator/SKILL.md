# NoiseCalculator - 噪声等效声级计算工具

## 功能概述

本工具基于GBZ/T189.8-2007标准，提供噪声等效声级的专业计算功能，包含以下计算模式：

### 单条计算模式

1. **全天等效声级计算**
   - 公式：L<sub>eq,d</sub> = 10×lg[(1/T<sub>总</sub>)×Σ(Tᵢ×10^(Lᵢ/10))]
   - 支持自定义多个时间段输入

2. **8小时等效声级计算**
   - 公式：L<sub>eq,8h</sub> = L<sub>eq,T</sub> + 10×lg(T/8)
   - 用于将实际工作时间的等效声级转换为8小时等效声级

3. **每周40小时等效声级计算**
   - 公式：L<sub>eq,40h</sub> = 10×lg[(1/5)×Σ(10^(L<sub>eq,i</sub>/10))]
   - 支持自定义每周工作天数

### 批量计算模式

支持批量处理多组数据，每行一组，格式如下：

- **类型1（全天等效声级）**: `1 声级1 时间1 声级2 时间2 ...`
- **类型2（8小时等效声级）**: `2 实际声级 实际工作时间`
- **类型3（每周等效声级）**: `3 工作天数 每日8小时声级1 每日8小时声级2 ...`

### 使用示例

单条计算：
- 设置时间段数量，填写各段声级和时间，点击计算按钮

批量计算示例：
```
1 80 4 85 6        # 类型1：2个时间段计算全天等效声级
2 88 12             # 类型2：实际声级88dB，工作12h
3 5 85 86 84 87 83  # 类型3：5个工作日计算每周等效声级
```
